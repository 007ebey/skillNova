Module 1: Excel Basics & Charts

Master structured data handling using Excel. Organize, clean, and represent information through bar, line, and pie charts that highlight trends and categories.

What Will You learn?

  • Create basic charts from datasets
  • Format and organize data into readable tables

Topics Covered

  • What is data and where does it live
  • Spreadsheet interface and cell navigation
  • Table formatting and sorting
  • Creating bar, pie, and line charts
  • Labeling and customizing chart elements
  • Interpreting visual patterns and trends

Module 2: Data Manipulation & Functions

Explore how Excel functions can automate insights. Use logical operations and formatting to reveal trends, totals, and patterns in school and real-world datasets.

What Will You learn?

  • Use formulas to extract meaning from data
  • Highlight patterns using smart formatting tools

Topics Covered

  • SUM, AVERAGE, MIN, MAX functions
  • IF statements and logical testing
  • COUNT and COUNTA
  • Conditional formatting rules
  • Filter and highlight based on criteria
  • Mini dashboards with multiple visuals

Module 3: Slide Design & Data Integration

Design clean, purposeful slides. Learn how to support storytelling with thoughtful layouts, color, and visual balance by integrating data visuals into effective presentation slides.

What Will You learn?

  • Structure slides that guide viewer attention
  • Embed charts and visuals to support a message

Topics Covered

  • Slide themes and layout options
  • Text formatting for readability
  • Inserting and positioning charts
  • Icons and SmartArt for storytelling
  • Using whitespace and alignment
  • Visual consistency in multi-slide decks
